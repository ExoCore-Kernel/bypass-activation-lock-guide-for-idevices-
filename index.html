Guide C

A5 Devices Method

This section is for A5 devices. The exact method, requirements, and troubleshooting content are placeholders for now.

A5 devices in this guide:
  • iPhone 4s
  • iPad 2
  • iPad mini (1st generation)
  • iPod touch (5th generation)

Requirements

  1. A Windows computer running Windows 10 or later
  2. Latest version of BobikA5
  3. A 30 pin to USB adapter

Refer to troubleshooting if you are unable to open the exe.

Steps

  1. Extract the ZIP file and run the BobikA5.exe application
  2. Connect your device to the computer
  3. Click on the 'Check device Compatibility button
  4. Once finished, click on the 'Activate your device' button and wait.
  5. Once complete, setup the device. You are done!

Do not setup a passcode when setting up the device as it will unactivate if you restart if a passcode is setup.

Troubleshooting

If you are unable to open BobikA5 try disabling Windows Defender, then try again.

Guide A

iOS 12.0 – iOS 14.8.1

This guide is for devices supported on iOS 12.0 through iOS 14.8.1.

Supported devices in this guide:
  • iPhone 6s
  • iPhone 6s Plus
  • iPhone SE (1st generation)
  • iPhone 7
  • iPhone 7 Plus
  • iPhone 8
  • iPhone 8 Plus
  • iPhone X
  • iPad Pro 12.9-inch (1st generation)
  • iPad Pro 12.9-inch (2nd generation)
  • iPad Pro 10.5-inch (1st generation)
  • iPad Air 2
  • iPad (5th generation)
  • iPad (6th generation)
  • iPad mini 4

Requirements

You will need:

  1. A Mac computer running macOS Monterey or later
  2. Latest version of Lockra1n
  3. A USB to Lightning cable (USB-C to Lightning cables may cause issues. Use USB-A to Lightning or a USB-A adapter if possible.)
Extra note: iPhone X on iOS 16 may be unstable and can have issues.

Steps

  1. Open the app you downloaded in the Requirements section. If you see an "Untrusted Developer" error, refer to troubleshooting.
  2. Click Prepare Lockra1n to make sure everything is installed and ready.
  3. Click Search for Device. Make sure to tap Trust on the device if prompted.
  4. Click Generate Activation Files. If you are not registered, the tool will prompt you to register.
  5. If you see Failed to generate tickets!, do not continue.
  6. Select Bypass Device (Normal mode). The Ramdisk method was deprecated.

To enter DFU Mode, the method varies depending on your device model. Search for your specific device model and DFU instructions.

The tool will complete its process and reboot your device. After your device turns back on, you should be on the Hello screen and be able to set it up as new.

Do not set up a passcode when setting up the device. If you reboot when the device is setup youll have to bypass again.

Troubleshooting

If macOS gives an Untrusted Developer error:

  1. Right-click the app file.
  2. Select Open.

If the app still will not open:

  1. Open Terminal.
  2. Type sudo xattr -cr.
  3. Drag the Lockra1n app into Terminal.
  4. Press Enter.
Guide B

iOS / iPadOS 15.0 and newer listed groups

This guide is for iOS / iPadOS 15.x, 16.x, 17.x, and 18.0 – 18.1.1.

Included version groups:
  • iOS / iPadOS 15.0 – 15.X.X
  • iOS / iPadOS 16.0 – 16.X.X

Supported devices across these newer groups:
  • iPhone 6s
  • iPhone 6s Plus
  • iPhone SE (1st generation)
  • iPhone 7
  • iPhone 7 Plus
  • iPhone 8
  • iPhone 8 Plus
  • iPhone X
  • iPad Pro 12.9-inch (1st generation)
  • iPad Pro 12.9-inch (2nd generation)
  • iPad Pro 10.5-inch (1st generation)
  • iPad Air 2
  • iPad mini 4
  • iPad (5th generation)
  • iPad (6th generation)
  • iPad (7th generation)

Requirements

You will need:

  1. A Mac computer running macOS Monterey or later
  2. Latest version of Lockra1n
  3. A USB to Lightning cable (USB-C to Lightning cables may cause issues. Use USB-A to Lightning or a USB-A adapter if possible.)

Steps

  1. Open the Lockra1n app you downloaded.
  2. If you see an Untrusted Developer message, follow the troubleshooting steps below.
  3. Click Prepare Lockra1n to install required components.
  4. Click Search for Device and press Trust on your device if prompted.
  5. Click Generate Activation Files.
  6. If you see Failed to generate tickets!, do not continue.
  7. Select Bypass Device (Normal mode). The Ramdisk method was deprecated.
  8. Enter DFU Mode for your device.

The tool will complete its process and reboot your device. After reboot, you should reach the Hello screen and youll be able to setup the device as normal.

Do not set up a passcode when setting up the device. If you reboot when the device is setup youll have to bypass again.

Important: The Re-Lock options do not fully remove all changes. To fully return the device to normal, use a clean restore. Do not use Update or Retain User Data.

Troubleshooting

If macOS blocks the app:

  1. Right-click the app.
  2. Click Open.

If it still will not open:

  1. Open Terminal.
  2. Type sudo xattr -cr.
  3. Drag the Lockra1n app into Terminal.
  4. Press Enter.

Extra Notes

iPhone X on iOS 16 may be unstable and can have issues.
